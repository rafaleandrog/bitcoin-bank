# FediBank v1.0

Production-style UI-first demo for Bitcoin wallet + lending + staking + RWA placeholder.

## Run
```bash
pnpm install
pnpm dev
```

## Test
```bash
pnpm test
pnpm test:e2e
```

## Highlights
- Federation-gated app access
- Lightning receive/send mocked flows
- BTC vault and BRL-yield stake positions
- BTC-collateralized loans with explicit no-liquidation-by-price rule
- Proof-of-reserves and institutional RWA landing placeholder
